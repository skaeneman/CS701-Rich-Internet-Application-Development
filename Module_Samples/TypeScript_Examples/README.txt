Installations
______________

Node.js and npm
  https://nodejs.org/en/download/
    Use the appropriate installer for your OS.

    
TypeScript
  http://www.typescriptlang.org/#download-links
  MAC:
     sudo npm install -g typescript
  Windows (From Command Window with Administrator Privelege):
     npm install -g typescript



Running the Typescript Samples

cd Basics
